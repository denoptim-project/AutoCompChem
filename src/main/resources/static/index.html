Key Features

  • Execute computational chemistry tasks via REST API
  • Upload and convert molecular structure files (SDF, XYZ, MOL, PDB)
  • Generate input files for Gaussian, Orca, NWChem, XTB, Spartan
  • Parse and analyze output files from quantum chemistry packages
  • Job evaluation and healing capabilities
  • Molecular geometry analysis and manipulation
  • Basis set generation and constraint handling
  • Conformational space exploration

Quick Start Examples

Get Available Tasks
GET /api/v1/autocompchem/tasks
Upload Molecular File
POST /api/v1/molecules/upload Content-Type: multipart/form-data Body: file=your_molecule.sdf
Generate Gaussian Input
POST /api/v1/compchem/software/gaussian/input Content-Type: application/json { "inputFile": "molecule.sdf", "outputFile": "gaussian.com", "method": "B3LYP", "basisSet": "6-31G*" }
Execute Custom Task
POST /api/v1/autocompchem/tasks/{taskName}/execute Content-Type: application/json { "parameter1": "value1", "parameter2": "value2" }

Getting Started

The AutoCompChem REST API provides a modern, web-based interface to the powerful computational chemistry automation capabilities of AutoCompChem. You can interact with the API using any HTTP client, programming language, or the interactive Swagger documentation.

To get started:

  1. Visit the Swagger UI to explore and test the API endpoints
  2. Check the available tasks endpoint to see what operations you can perform
  3. Upload a molecular structure file to test the file I/O capabilities
  4. Generate input files for your preferred quantum chemistry software
